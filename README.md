## 📂 Projetos

### ☕ Cafeteria API
API REST completa para gerenciamento de uma cafeteria, implementando:
- Operações **CRUD** (Create, Read, Update, Delete)
- Persistência de dados com Entity Framework
- Boas práticas de arquitetura em camadas (MVC)

```
├── Cafeteria/          # Projeto principal da API
├── CafeteriaAPI/       # Configuração da API
└── BackEnd - 2º Semestre.sln  # Solution file
```

```bash
dotnet run
```
